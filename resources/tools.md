# AI Tools Directory

A curated list of AI tools relevant to academic teaching and research. Tools are grouped by primary use case. Most have free tiers adequate for course activities.

> **Note:** The AI tools landscape changes rapidly. Check each tool's website for the most current features, pricing, and data privacy terms.

---

## General-Purpose Large Language Models

These are the most flexible AI tools — useful for writing, brainstorming, explaining, summarising, and many other tasks.

| Tool | Provider | Notes |
|------|----------|-------|
| [ChatGPT](https://chat.openai.com) | OpenAI | Most widely used LLM. GPT-4o available on free tier. |
| [Claude](https://claude.ai) | Anthropic | Strong at long documents, nuanced writing, and safety. |
| [Gemini](https://gemini.google.com) | Google | Integrates with Google Workspace. Multimodal. |
| [Microsoft Copilot](https://copilot.microsoft.com) | Microsoft | Integrated with Office 365; uses GPT-4. |
| [Mistral Le Chat](https://chat.mistral.ai) | Mistral AI | European provider with strong privacy stance. |
| [Perplexity AI](https://www.perplexity.ai) | Perplexity | Web-grounded search with citations; reduces hallucination risk. |

## Choosing the Right Tool

Use this quick guide:

| I want to... | Start with... |
|-------------|--------------|
| Explore AI for the first time | ChatGPT or Claude (free tiers) |
| Find academic papers on a topic | Elicit or Consensus |
| Read and interrogate a specific paper | NotebookLM |
| Improve my academic writing | Writefull or Grammarly |
| Analyse data without coding expertise | Julius AI or ChatGPT with Code Interpreter |
| Screen abstracts for a systematic review | Rayyan |
| Work with sensitive/confidential data | Ollama (local) or institutional enterprise plan |
| Create a lecture or presentation quickly | Gamma |
| Transcribe a lecture or interview | Otter.ai |
